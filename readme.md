# プログラミング実習II 自動採点プログラム
## 使い方
1. `git clone [このリポジトリのURL]`
2. default-config.ini のファイル名を config.ini に変更
   1. config.ini の中のパラメータを設定
3. prog2checker.exe を実行

### Linux の場合
- 環境セットアップ
  - `python -m venv [仮想環境名]`
  - 仮想環境を activate する
  - `pip install -r requirements.txt`
- 起動
  - 仮想環境を activate する
  - `python prog2checker.py`
